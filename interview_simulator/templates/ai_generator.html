How to Enable Groq

  1. Set GROQ_API_KEY in environment.
  2. Set model to grok-4-1-fast-non-reasoning.
  3. Set base URL to https://api.groq.com/openai/v1.
  4. Use this page to generate category-wise MCQs instantly.

Examples

GROQ_API_KEY=your_key_here

GROQ_MODEL=grok-4-1-fast-non-reasoning

GROQ_API_BASE_URL=https://api.groq.com/openai/v1
